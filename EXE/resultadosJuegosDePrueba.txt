Este documento contiene los teclados generados a partir de los distintos 
juegos de prueba que hemos creado. 

¡TODOS LOS RESULTADOS GENERADOS SÓN CON ALFABETOS PERSONALIZADOS!
Excepto los que se indican que no lo son.
Esto es porque si se escoge un idioma como alfabeto y no se usan
todas las letras la variación entre los posibles teclados resultantes
es muy alta.


Juego de prueba: textoCortoCatalan.txt
Objeto de la prueba: comprobar que un texto corto en catalan funciona correctamente
Ficheros de datos necesarios: habrá que crear una entrada con este texto que se
                              guardara automáticamente en Entradas.ser
Valores estudiados: Caja negra. Se estudia el resultado de la ejecución del programa
                    con el texto de prueba.
Efectos estudiados: Indirectamente se estudiará que los menus funcionan correctamente
                    ya que se usan para seleccionar el texto de prueba.
Operativa: Se ejecuta el programa, se creará una entrada importando textoCortoCatalán y se
           usará solamente esa entrada para crear un teclado con alguno de los dos algoritmos
           y con el alfabeto personalizado. También se comprobará que se puede escoger el catalán
           como alfabeto.

posibles resultados:
QAP:
c a l ç o t s

Annealing:
c a l ç o t s
